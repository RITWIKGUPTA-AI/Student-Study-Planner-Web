# Student Study Planner

A Flask-based study management web application for students.

## Features

- User Login
- Study Planner
- Task Management
- Notes
- Goals
- Timetable
- Attendance Tracking
- Analytics
- Reminders
- Backup System

## Technologies

- Python
- Flask
- HTML
- CSS
- Bootstrap
- JSON Storage

## Installation

Clone repository:

git clone YOUR_LINK


Install requirements:

pip install -r requirements.txt


Run:

python app.py
